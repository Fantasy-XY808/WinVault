## ✨ Features

### 🖥️ **System Information**
- **Hardware Detection** - Detailed CPU, memory, GPU, and storage information
- **Real-time Monitoring** - Temperature, usage rates, and performance metrics
- **System Reports** - Generate comprehensive system configuration reports

### ⚙️ **Service Management**
- **Service Control** - Start, stop, and restart Windows services
- **Dependencies** - View service dependencies and relationships
- **Startup Types** - Modify service startup types and configurations

### 🔧 **Utilities**
- **Command Tools** - Graphical interface for common CMD and PowerShell commands
- **Quick Settings** - One-click access to Windows system settings
- **System Cleanup** - Clean temporary files and system junk

### 🎨 **Interface Features**
- **Modern Design** - Based on WinUI 3 Fluent Design
- **Responsive Layout** - Adapts to different screen sizes
- **Theme Support** - Light/dark theme switching

## 📥 Quick Start

### Download & Install

**Method 1: Direct Download (Recommended)**
1. Visit [Releases page](https://github.com/Fantasy-XY808/WinVault/releases)
2. Download the latest `WinVault-Setup-x64.exe`
3. Run the installer and follow the prompts

**Method 2: Build from Source**
```bash
# Clone the project
git clone https://github.com/Fantasy-XY808/WinVault.git
cd WinVault

# Build the project
dotnet restore
dotnet build -c Release

# Run
dotnet run -c Release
```

### System Requirements

| Item | Minimum | Recommended |
|:---:|:---:|:---:|
| **OS** | Windows 10 1809+ | Windows 11 |
| **CPU** | x64 Dual-core 1.8GHz | x64 Quad-core 2.4GHz+ |
| **RAM** | 2GB RAM | 4GB+ RAM |
| **Storage** | 100MB free space | 500MB+ |
| **Runtime** | .NET 8.0 Runtime | .NET 8.0 SDK |
